# Ask Genie

**Ask Genie** turns your data-quality results into a conversation. Open the panel
on any [results page](/docs/studio/running/results) and ask a question in plain
English — "which columns have the most failures?", "why did my score drop?" — and
Genie answers with a short explanation and, where it helps, a chart. There's no
SQL to write and no query to build; it's built on Databricks Genie and reads the
same results you're already looking at.

It's the fastest way in for a steward or reviewer who wants an answer without
digging through the breakdowns by hand. Every conversation stays with the table
or collection you opened it from, and you can start a **New conversation** or
revisit an earlier one from **History** at any time.

## Start with a suggested question

Not sure what to ask? Genie opens with a set of **suggested questions** grouped by
what you're trying to do — **Basic stats**, **Drilldown**, **Trends over time**,
and **Diagnose**. Pick one to get going, or type your own in the box at the
bottom.

## Explain a change over time

Ask it to diagnose a shift in the score, and it breaks down which rules moved and
by how much:

> What is driving my changes in score over time?

Genie compares the runs and shows the per-rule before-and-after fail rates, so you
can see exactly which check pulled the score up or down.
